# inventoryManage

A C/C++ console-based inventory management program that allows users to add, update, search, and remove items while tracking quantities and item details.

## Features
- Add new inventory items (name/ID/quantity)
- Update item information and quantities
- Search for items in the inventory
- Remove items from the inventory
- Input validation to prevent invalid entries

## Tech Stack
- C / C++

## How to Run
1. Compile:
   ```bash
   g++ main.cpp -o inventory
   ./inventory
